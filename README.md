# Python Data Structures and Algorithms

A comprehensive repository for learning and practicing various Data Structures and Algorithms implemented in Python.

## 🚀 Overview

This repository is logically organized by data structure and algorithm topics. It serves as a learning resource and a valuable reference for common algorithmic problem-solving patterns and foundation data structure principles.

## 📁 Repository Structure

- **`Array/`** \- Array-based questions and solutions.
- **`Graph/`** \- Graph traversal algorithms (BFS, DFS) and related concepts.
- **`Hash Table/`** \- Basics and implementations involving collision resolution and usage.
- **`Linked List/`** \- Single, Double, and Circular Linked Lists logic and algorithms.
- **`Queue/`** \- Implementations of basic Queues, Dequeues, and more.
- **`Recursion/`** \- Problems exploring the fundamentals of recursion and backtracking.
- **`Search/`** \- Searching implementations like Linear Search and Binary Search.
- **`Sort/`** \- Sorting algorithms including Bubble, Insertion, Selection, Merge, and Quick Sort.
- **`Stack/`** \- Various stack operations and relevant interview questions.
- **`String/`** \- String manipulation algorithms.
- **`Tree/`** \- Implementations encompassing Binary Search Tree (BST), Binary Trees, and traversals.
- **`Trie/`** \- Prefix tree implementation and text processing.
- **`Workouts/`** \- Experimental practice files and rough work routines.
- **`workout.py`** \- A standalone script for quick testing and drafting out algorithms.
- **`main.py`** \- Available as a general playground or entry point to run modules.

## 🛠️ Requirements

- **Python 3.x**: Ensure you have a relatively recent version of Python installed.

## 📜 How to run

Clone the repository to your local machine:
```bash
git clone <your-repository-url>
```

Run any python file using the standard python command line execution:
```bash
python <path-to-file>.py
```
For example, to run an array problem script:
```bash
python Array/some_array_file.py
```
